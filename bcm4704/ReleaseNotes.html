What's New in Release 4.150.10.24?

  • "GURR++" release
  • Updated DFS Support
  • Router Secure IBSS added (WPA-PSK like link over AdHoc)
  • Updates to support WiFi WPS certification
  • Fix issue with MBSS Probe Response transmission when rate fixed at a legacy rate
  • Fix for compressed CFE on 5354 A3 designs
  • Various bug fixes and data path optimizations

What's New in Release 4.150.10.15?

  • "GURR+" release
  • Driver support for Single Stream N prototyping
  • Per-SCB statistics added
  • Fix for WiFi 11n Countermasures failure on 4321 B1 chips
  • Disable 5600MHz-5650MHz in Canada Locale
  • Various WPS fixes and features, including support for AP Lockdown
  • Fix for VxWorks specific occasional WPA authentication failures
  • Fix for RSSI/GainBoost for 4321 in 5GHz
  • Fix Efficient Multicast Forwarding not working when endpoints configured for Access Point mode

What's New in Release 4.150.10.7.?

  • "GURR" release for VxWorks
  • VxWorks-specific performance improvements compared to 4.150.10.5

What's New in Release 4.150.10.5?

  • "GURR" release for Linux
  • Support for  4-BSS MBSS on 4318, 4321, and 5354 platforms
  • Support for WPS and Microsoft VistaRouter (Basic) Certification
  • Efficient Multicast Forwarding feature added
  • QoS - Retry Limits and Upper Rate configurable per Access Class
  • MAC Address Spoofing for WET (STA) mode
